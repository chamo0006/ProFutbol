# ProFutbol-UADE ⚽

Proyecto académico desarrollado en la Universidad Argentina de la Empresa (UADE).  
Trabajo práctico grupal orientado al desarrollo de un sitio web temático de fútbol, con funcionalidades de foro, tienda y noticias.

## 🚀 Tecnologías utilizadas
- **HTML5** para la estructura de las páginas
- **CSS3** para estilos y diseño responsivo
- **JavaScript** para interactividad y lógica de la tienda/foro

## 📂 Funcionalidades principales
- **Foro de usuarios**: registro, login y publicación de mensajes
- **Tienda online**: carrito de compras y gestión de productos
- **Noticias deportivas**: sección dinámica con contenido actualizado
- **Diseño responsivo**: adaptable a distintos dispositivos
